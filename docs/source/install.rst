Install
============

Using pip install:

.. code-block:: console

    # Stable release (recommended)
    pip install -q "harvest_plet @ git+https://github.com/DTO-BioFlow/DUC3_harvest_plet_data@v0.2.0"

    # Latest development branch
    pip install -q "harvest_plet @ git+https://github.com/DTO-BioFlow/DUC3_harvest_plet_data@main"
